# LAUNCH-TOP-5-ASSET-BRIEFS-AND-PROMPTS

Prompt pack for the first 5 launch episodes.

Use this to generate:
- hero food images
- thumbnail / cover concepts
- blurred background plates
- icon style references
- optional pixel-art sprite references

Designed for:
- Midjourney
- Flux
- DALL-E
- Stable Diffusion
- similar image tools

General rule:
- no text in generated images
- no hands unless specifically requested
- no extra plates/utensils unless needed
- optimize for vertical-safe composition
- keep food identity obvious at phone size

---

# GLOBAL STYLE BASELINE

## Hero image baseline prompt
Use this structure for all main food heroes:

`[FOOD], polished food hero shot, centered composition, vertical 9:16 safe framing, clean silhouette, strong readable texture, cozy game-like realism, slightly stylized but believable, soft controlled lighting, rich but not overprocessed color, subtle depth, background simple enough for UI overlay, no text, no watermark, no hands, no clutter`

## Thumbnail baseline prompt
`[FOOD], thumbnail-ready food hero, instant recognition at phone size, bold silhouette, strong center framing, clean contrast, dramatic but tasteful lighting, background simplified for title and verdict overlay, cozy game-like realism, no text, no watermark, no hands, no clutter`

## Background plate baseline prompt
`abstract background plate inspired by [FOOD], low-detail blurred forms, soft gradients, cozy retro-game palette, subtle texture, designed to sit behind UI, no text, no objects that fight the foreground, vertical 9:16`

## Sprite reference baseline prompt
`[FOOD], pixel-art sprite reference, front-facing or 3/4 angle, simplified readable shapes, cozy retro RPG item style, clear silhouette, no text, transparent-background-friendly look, charming but not childish`

## Negative prompt baseline
Use where supported:
- text
- watermark
- logo
- human hands
- extra utensils
- messy table clutter
- weird anatomy
- mushy texture
- muddy colors
- overexposed highlights
- extreme depth blur
- stock-photo wellness vibe

# EXPORT / REVIEW CHECKLIST

Before locking any generated asset:
- instantly readable at phone size
- no text embedded in image
- subject centered enough for 9:16 hook frame
- enough empty room for header + subtitles + verdict overlay
- food looks appetizing or appropriately tempting
- not too photoreal ad-like, not too childish
- style remains consistent with cozy game-like system
