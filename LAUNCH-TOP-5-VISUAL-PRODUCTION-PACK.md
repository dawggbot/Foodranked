# LAUNCH-TOP-5-VISUAL-PRODUCTION-PACK

This pack turns the first 5 launch candidates into build-ready visual episode specs.

Use this as the bridge from:
- scored/scripted episode package

to:
- actual short-form visual production

---

## Global visual rules for the first launch batch

### Format
- 9:16 vertical
- use `templates/visual-template.v1.json` as the default base
- keep the header visible and stable through the whole video
- let the spoken script carry the explanation; visuals should support, not compete

### Header contents
Header should always show:
- food image / hero crop
- food name
- food type
- basis (`per 100g`)
- kcal

Do not speak that information unless needed for clarity.
It is visible context.

### Subtitle rules
- subtitles are the only body text layer
- max 2 lines in normal use
- use 3 only in emergency moments
- lift subtitle block upward when pros/cons bullets are on screen
- never cover the tier stamp
- keep subtitles centered and large enough for phone speed

### Motion rules
- one major reveal per spoken clause
- no floaty over-animated transitions
- crisp card pops, stamp hits, and stat flips only
- use subtle 8-bit/cozy UI sounds on each major reveal
- the verdict stamp must be the biggest motion beat in the video

### Progress indicator
Use the dot row as a true pacing cue.
Recommended active-dot flow:
1. hook
2. fats
3. carbs
4. proteins
5. micros
6. pros/cons
7. verdict

### Asset baseline per episode
Each episode should have:
- hero food image
- optional sprite / icon cutout
- blurred color background derived from the food palette
- one thumbnail frame candidate
- one tier reveal frame candidate

# 1) BACON

## Why this is a strong opener
- extremely legible food
- immediate emotional recognition
- negative verdict creates comment energy
- easy to understand visually within 1 second

## Visual posture
- category accent: deep red
- overall mood: smoky, high-contrast, slightly dangerous
- make it feel appetizing at first, then let the UI slowly turn against it

## Hook frame (0.0–2.4s)
Show:
- large bacon hero stack, slightly glossy, centered
- header already filled
- `BACON` title hit
- `RANKED` pops a beat later

Motion:
- bacon hero scales in slightly
- `RANKED` should feel like a heavy arcade drop, not playful bounce

## Fats section (2.4–10.6s)
Narration focus:
- high fat
- saturated fat is a drawback
- fat quality matters for meats

Show:
- macro headline: `42g fat`
- submetric cards:
  - saturated fat
  - polyunsaturated fat
- use a red-to-purple warning read on the saturated-fat card

Direction:
- this is the first tonal turn downward
- use a subtle warning pulse when saturated fat appears

## Carbs section (10.6–19.3s)
Narration focus:
- carbs do not matter here
- meats live or die on protein + fat profile

Show:
- keep this section visually lighter than fats/protein
- macro headline `1.4g carbs`
- dimmed side cards or minimal placeholders

Direction:
- treat this as a quick bridge, not a hero section
- visually communicate “not the main story”

## Proteins section (19.3–27.5s)
Narration focus:
- strong protein quality
- protein alone does not save bacon

Show:
- macro headline `37g protein`
- strong green/blue lift for protein quality cards

Direction:
- this section should briefly feel like bacon is recovering
- important: do not let it feel like a fake win; the later cons need to overpower it

## Micros block (27.5–44.4s)
Narration focus:
- B12 exists
- vitamins/minerals are not enough to save it

Show:
- vitamin rows first, then minerals
- use smaller energy than macro sections
- avoid making tiny DV values feel impressive

Direction:
- visual tone should be “some support, but not enough”
- do not over-celebrate B12

## Pros block (44.4–59.0s)
Narration focus:
- tastes good
- works as flavour booster
- strong comfort-food appeal

Show:
- 3 green bullets
- each bullet gets a quick icon:
  - taste/sizzle
  - flavour boost/pan spark
  - comfort/heart-steam

Direction:
- this section should feel socially honest
- let viewers feel “yes, fair enough” before the cons hit

## Cons block (59.0–76.3s)
Narration focus:
- sodium
- processing
- fat profile

Show:
- 3 red bullets
- heavier reveal pacing than pros
- last bullet should land hardest

Direction:
- use this as the true downhill slide into the verdict
- strongest beat: processing + category mismatch

## Verdict (76.3–79.0s)
Show:
- huge D-tier stamp
- purple impact ring
- score plate `39`

Direction:
- verdict should feel blunt, not tragic
- this is a “yeah, that tracks” punchline

## Thumbnail candidate
- bacon strips in center
- big D stamp partially covering one side
- header visible
- strong contrast between appetizing image and bad verdict

## Asset notes
Need:
- hero bacon stack image with strong texture
- optional pixel sizzle particles
- 3 tiny icons for pros and 3 harsher icons for cons

- stacked rice cakes, bright clean look
- slightly over-wholesome opening aesthetic

Direction:
- the hook should almost imply “this might be good”
- later sections should peel that back

## Fats section
Narration focus:
- not a fat story

Show:
- minimal energy
- tiny fat value with low dramatic weight

Direction:
- keep it moving fast

## Carbs section
Narration focus:
- huge carbs
- harsh glycemic impact
- weak payoff

Show:
- macro headline `81.5g carbs`
- glycemic index card gets strongest emphasis
- starch card appears second

Direction:
- this is the core section of the episode
- use a visible “spike” visual motif to suggest fast blood-sugar rise without overcomplicating it

## Protein section
Narration focus:
- protein is too weak to rescue it

Show:
- modest protein card
- no heroic color treatment

Direction:
- treat protein as a failed rescue attempt visually

## Micros block
Narration focus:
- basically weak support

Show:
- sparse-looking vitamin/mineral rows
- empty-feeling spacing is okay here

Direction:
- make the low support feel visually obvious

## Pros block
Narration focus:
- portable
- easy to carry
- easy with toppings

Show:
- 3 practical utility bullets
- icons: bag / hand carry / topping-plus

Direction:
- this should feel practical, not nutritionally impressive

## Cons block
Narration focus:
- weak grain payoff
- glycemic impact
- poor fullness

Show:
- 3 red bullets with more weight than the pros
- last bullet should feel especially relatable: “not filling enough”

Direction:

## Verdict
Show:
- D-tier stamp

## Thumbnail candidate
- clean rice cake stack
- big D stamp
- subtitle-style mini line optional during cover creation only: `healthy-looking, weak payoff`

## Asset notes
Need:
- strong clean rice cake hero image
- one glycemic spike icon/motion motif
- practical icons for portability/toppings

# Production checklist for each episode

## Editorial
- lock final narration draft
- confirm verdict wording
- confirm score line wording

## Visuals
- choose hero food image
- choose palette treatment
- choose subtitle-safe layout test frame
- create bullet icons if needed

## Motion
- map narration clauses to reveal slots
- ensure tier stamp lands exactly on verdict phrase
- test subtitle overlap on pros/cons sections

## Export QA
- phone-size readability check
- subtitle density check
- no body text beyond subtitles
- verdict frame readable within 0.5 seconds
