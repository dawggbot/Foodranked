# Context Rewrite Drafts

These are draft replacement directions for foods whose current pros/cons are too generic.
The aim is to make every pro/con slot earn its place.

---

## Yam

### Better candidate pros
1. **useful fibre for a starch-heavy root**
   - Helps yam separate from lower-fibre carb staples.
2. **brings a decent potassium return**
   - Better than empty-starch options if canonical values hold.
3. **solid whole-food staple value**
   - Still practical, but framed as a whole-food strength rather than vague category filler.

### Better candidate cons
1. **nutrition profile depends on exactly which yam is meant**
   - Important because yam data is commonly confused with sweet potato.
2. **micronutrient return is not automatically elite**
   - Needs evidence before claiming exceptional status.
3. **still mainly an energy food**
   - Fairer and more specific than generic category filler if phrased as a ceiling, not a throwaway.

### Special-compound / exception shortlist
- carotenoids? only if the actual yam profile supports it
- resistant starch potential depending on preparation/cooling
- digestion/tolerance differences by preparation

## Turnip

### Better candidate pros
1. **lighter root option with low calorie load**
   - Distinguishes it from denser tubers.
2. **useful vitamin C contribution for the calories**
   - More specific than generic usefulness claims.
3. **clean whole-food flexibility**
   - Keep one practicality point, but not three.

### Better candidate cons
1. **not a strong staple-carb performer**
   - More honest than pretending it is a major energy anchor.
2. **fibre return is only modest**
   - Distinguishing weakness.
3. **micronutrient profile is respectable, not category-leading**
   - Better than vague carb-category boilerplate.

### Special-compound / exception shortlist
- glucosinolate/cruciferous angle depending on whether root-only framing makes that appropriate
- bitterness / taste barrier if it becomes a real practical downside worth mentioning

---

## White Rice

### Better candidate pros
1. **easy to digest for many people**
   - More meaningful than generic batch-cooking praise.
2. **very practical staple base**
   - Keep one practicality point because it genuinely matters.
3. **works well in low-fibre / bland-food situations**
   - More food-specific and useful than generic meal-anchor copy.

### Better candidate cons
1. **fibre is very weak for a grain**
   - Core category failure.
2. **glycemic behaviour is rougher than stronger grains**
   - Big differentiator.
3. **refinement strips away much of the grain’s upside**
   - Better than generic processing wording.

### Special-compound / exception shortlist
- low residue / low-fibre tolerance use case
- lower phytate burden than some whole grains, if worth noting

## Whole Wheat

### Better candidate pros
1. **strong fibre density for a staple grain**
2. **much better mineral return than refined grains**
3. **steadier whole-grain carb package**

### Better candidate cons
1. **phytates can slightly reduce mineral uptake**
2. **not suitable for gluten-sensitive people**
   - Must be used carefully; not a universal con, but potentially still notable.
3. **can lose a lot of its edge in heavily processed forms**
   - Important if the ranked item is broad whole-wheat, not a pristine kernel.

### Special-compound / exception shortlist
- whole-grain integrity
- bran/germ retention
- phytate/mineral absorption nuance

## Tomato

### Better candidate pros
1. **lycopene is the obvious standout**
2. **high volume for very few calories**
3. **easy low-downside everyday use**

### Better candidate cons
1. **fibre is modest compared with stronger vegetables**
2. **micronutrient density is decent, not elite**
3. **acidity can be annoying for some people**
   - Use only if that beats weaker generic cons.

### Special-compound / exception shortlist
- lycopene
- improved lycopene availability in cooked/processed tomato forms
- acidity/reflux tolerance nuance

## Zucchini

### Better candidate pros
1. **excellent volume-for-calories trade**
2. **easy way to add bulk to meals without much downside**
3. **very easy to integrate into regular cooking**

### Better candidate cons
1. **fibre is only modest**
2. **micronutrient density is not especially standout**
3. **brings less distinctive nutritional upside than elite vegetables**

### Special-compound / exception shortlist

## Kale (cons cleanup)

### Stronger candidate cons
1. **flavour/texture can be a barrier for some people**
2. **prep matters more than with easier vegetables**
3. **very high vitamin K can matter for some medication contexts**
   - Use carefully and only if the project wants practical edge-case cautions.

### Special-compound / exception shortlist
- glucosinolates
- carotenoids/lutein/zeaxanthin
- vitamin K density
- oxalate/goitrogen nuance depending on desired detail level

## Oats (polish)

### Best likely pros
1. **beta-glucans are a genuine standout**
2. **strong fibre + mineral package for a grain**
3. **may have unusual extra compounds like avenanthramides**
   - Keep only if evidence threshold is met.

### Best likely cons
1. **phytates can slightly reduce mineral uptake**
2. **some people do not tolerate oats particularly well**
3. **common toppings can easily ruin the clean baseline**
   - Keep only if we want practical-use framing, not pure food-only framing.

### Special-compound / exception shortlist
- beta-glucans
- avenanthramides
- satiety advantage
- phytate caveat

---

## Notes on style
- Avoid generic filler like "easy to pair" unless it is doing real comparative work.
- Avoid category boilerplate like "still a carb-heavy category" unless it is tied to a real differentiating weakness.
- Prefer one practicality point max, not three copies of the same idea.
- Top-3 slots should go to things that genuinely distinguish the food from peers.
