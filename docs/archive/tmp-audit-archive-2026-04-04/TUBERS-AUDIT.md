# Tubers Audit

## Foods reviewed so far
- white-potato
- sweet-potato
- yam
- fries
- turnip

## Category philosophy
Tubers should reward:
- whole-food carb quality
- fibre
- steadier glycemic behaviour
- useful vitamin/mineral return
- practical staple value

Tubers should penalize:
- fried / heavily processed formats
- harsh glycemic behaviour
- needless energy-density inflation

Protein should matter very little.
Fat should matter very little unless preparation makes it a real problem.

## Current strengths
- Sweet potato is already shaped to sit high.
- White potato is framed as respectable, not elite.
- Fries are clearly framed as a category failure case.
- Ruleset emphasis broadly matches the tuber calibration philosophy.

## Current problems
- Yam context is too generic and category-level.
- Turnip context is also too generic and not specific enough to turnip.
- Several context items are filler and don’t really deserve top-3 status.
- Generic lines like "still a carb-heavy category" are weak cons because they do not distinguish one tuber from another.
- Some food-specific opportunities are being missed:
  - sweet potato -> carotenoids / vitamin A clearly deserve priority
  - white potato -> satiety / potassium / resistant starch potential depending on preparation may matter
  - yam -> species confusion risk means identity must be clarified before strong claims
  - turnip -> may belong closer to a lower-starch, lower-energy root vegetable framing than a classic dense tuber framing

## Food-by-food notes

### Sweet potato
Likely keep high.
Good current positive:
- carotenoid bonus

Weak current cons:
- protein contribution is light -> probably too generic to matter here
- prep dependency -> okay as a caveat, but should not consume a top-3 con slot unless we lack better negatives
Possible stronger framing:
- very strong provitamin A / carotenoid story
- useful fibre and gentler GI vs harsher tubers
- whole-food staple versatility
Potential better cons:
- sweeter profile / higher natural sugars than some peers
- still calorie-bearing / carb-dense enough to overdo if the use case is misaligned
- preparation can ruin the advantage when turned into dessert-style or heavily fat-loaded formats

### White potato
Current placement around B feels plausible.
Best likely pros:
- practical staple value
- decent satiety when minimally processed
- useful potassium
Weak current con:
- micronutrient return is fine rather than elite -> okay, but could be sharper
Possible stronger nuance:
- cooled potato / resistant starch angle may matter as an exception note, but should be used carefully because the ranked baseline appears to be generic white potato, not specifically cooled potato.

### Yam
Needs rewrite.
Current pros/cons are too category-generic.
Need to clarify exactly which yam nutrient profile is intended because yam vs sweet potato confusion is common.
USDA anchor identified in this pass: `fdcId 170071` / `ndbNumber 11601` for `Yam, raw` (`Dioscorea spp.`), with useful nutrient-row anchors like energy `foodNutrientId 1492573`, protein `1492608`, potassium `1492586`.
That improves auditability a lot, but does not fully solve the broader species/market-label ambiguity.
Potential notable positives if supported by canonical sources:
- fibre
- potassium
- vitamin C
- specific carotenoid value depending on yam type
Potential notable negatives:
- species inconsistency / data ambiguity
- still mainly an energy food, not a micronutrient monster unless the specific yam profile supports it

### Fries
Current direction is right.
Strong cons are appropriate:
- fried format penalty
- rough glycemic behaviour
- harsh energy density
Current pros are weak and too pleasure-oriented.
"comfort-food appeal" and "satisfying texture" are real-world true, but they are not nutrition-forward and probably should not affect scoring.
Likely better approach:
- keep non-scoring editorial flavour for narration if desired
- use only one restrained positive like "still retains some potato micronutrients" if needed
- avoid letting hedonic appeal distort food scoring

### Turnip
Current context is almost all filler.
It reads like a generic tuber template.
Potential better direction:
- lower-calorie root/tuber-like whole-food option
- useful vitamin C / lighter energy load
- practical but less category-defining as a staple carb
Potential cons:
- not very energy-dense or satiating compared with true staple tubers
- fibre and mineral return are not elite
- may be less versatile / less appealing as a main carb anchor depending on use case
