# Vegetables Audit

## Foods reviewed so far
- kale
- tomato
- zucchini

## Category philosophy
Vegetables should reward:
- micronutrient density
- fibre where meaningful
- very low downside whole-food quality
- useful phytonutrient/plant-compound advantages
- low glycemic burden

Vegetables should penalize:
- weak micronutrient return relative to peers
- low fibre relative to peers
- only modest standalone usefulness where relevant

## Current strengths
- Kale is correctly framed as an elite vegetable candidate.
- Tomato and zucchini are at least recognized as low-downside, easy-to-use vegetables.
- Ruleset is correctly vitamin/mineral dominant.

## Current problems
- Tomato and zucchini context items are extremely repetitive and generic.
- Even kale includes some weak filler cons like "rarely carries a meal by itself," which is not very helpful because it applies to almost every vegetable.
- The system is not yet using the best vegetable-specific exception opportunities:
  - kale -> carotenoids / glucosinolates / vitamin K density
  - tomato -> lycopene
  - zucchini -> hydration / volume / lower standout micronutrient ceiling

## Food-by-food notes

### Kale
Should remain very high.
Potential best positives:
- enormous vitamin K density
- strong vitamin C / vitamin A support
- cruciferous/glucosinolate angle if evidence framing is wanted
Potential better cons:
- oxalate/goitrogen/digestive bitterness/prep friction depending on how nuanced we want to be
Current generic cons about not carrying a meal alone are weak and should probably be replaced.

### Tomato
Needs much more food-specific framing.
Potential best positives:
- lycopene standout
- high volume for very low calories
- easy practical use / low downside
Potential better cons:
- micronutrient density is modest vs elite vegetables
- acidity/tolerance issues for some people if worth noting
- lower fibre than stronger vegetables
Current cons are too repetitive and not food-specific.

### Zucchini
Current context is almost pure generic template text.
Potential positives:
- high-volume low-calorie utility
- easy integration into meals
- hydration / low downside
Potential cons:
- modest fibre
- modest micronutrient density
- not much category-leading upside beyond being easy to use

## Vegetables likely rewrite priority
1. tomato
2. zucchini
3. kale cons cleanup
