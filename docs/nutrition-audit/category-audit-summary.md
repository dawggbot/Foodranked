# Category Audit Summary

This summary distills the most useful reasoning from the temporary audit notes into a permanent reference.

## Cross-category rules

Before claiming a food is production-safe:
- lock the exact food identity and form
- verify the per-100g basis for the ranked form
- attach provenance for canonical values
- check whether the ruleset is truly tailored to the food type
- check whether the food belongs in its tier relative to peers
- ensure pros/cons are food-specific and not generic filler

## Grains

### Philosophy
Reward:
- fibre density
- better glycemic behaviour
- whole-grain integrity
- useful mineral return
- practical staple value

Penalize:
- refinement
- low fibre
- harsher glycemic behaviour
- weak micronutrient return

### Current watchouts
- white rice is easy to over-credit for practicality
- oats deserves exception-aware handling for beta-glucans and possibly avenanthramides
- whole wheat should clearly outrank refined grains, but context needs to avoid generic filler

## Tubers

### Philosophy
Reward:
- whole-food carb quality
- fibre
- steadier glycemic behaviour
- useful vitamin/mineral return
- practical staple value

Penalize:
- fried/heavily processed formats
- harsh glycemic behaviour
- needless energy-density inflation

### Current watchouts
- yam labeling must stay explicit so it does not drift back into sweet-potato confusion
- sweet potato deserves carotenoid/provitamin A priority
- white potato should be judged as respectable, not magically elite
- turnip needs more specific treatment and may sit closer to a lighter root-vegetable framing than a classic staple tuber framing

## Vegetables

### Philosophy
Reward:
- micronutrient density
- meaningful fibre
- very low downside whole-food quality
- useful phytonutrient advantages
- low glycemic burden

Penalize:
- weak micronutrient return relative to peers
- low fibre relative to peers
- modest standalone usefulness when that distinction genuinely matters

### Current watchouts
- kale should rely on real strengths like vitamin K, carotenoids, and glucosinolate-adjacent framing rather than filler
- tomato deserves lycopene-specific treatment without being overinflated into elite territory
- zucchini should be framed honestly as low-downside/high-volume, not as a hidden micronutrient monster

## Food-specific compound reminder list

Use these only when genuinely notable for the food:
- beta-glucans
- resistant starch
- polyphenols
- flavonoids
- avenanthramides
- carotenoids
- glucosinolates
- anthocyanins
- lycopene
- lutein / zeaxanthin
- selenium richness
- omega-3 form quality
- amino acid quality
- unusually good bioavailability

## Context quality rule

Pros/cons should not just restate macro, vitamin, or mineral points already shown on screen.
They should add context like:
- food-specific compounds
- bioavailability
- anti-nutrients
- digestion/tolerance
- satiety
- processing burden
- preparation traps
- sourcing/authenticity when materially relevant

For production use, the highest-value context items should also be classified by evidence strength:
- `supported`
- `plausible`
- `editorial`
- `contested`

Scoring context should lean toward `supported`.
Purely editorial lines should not quietly influence rank.

## Highest-risk next audit targets

### Foods
1. **tomato** — cleanest near-finish candidate; raw/cooked/sauce/paste separation is enforced in-file, and the next real need is citation hardening rather than identity rescue
2. **oats** — strong near-finish candidate, but still wants a better rolled-oats-specific primary source if available
3. **white rice** — current canonical direction is cleaner now, but GI/citation hardening still matters because rice behaviour is preparation-sensitive
4. **chicken thigh** — canonical wholefood entry should now be raw, and the main remaining work is locking a trustworthy raw source row
5. **yam** — canonical direction is now true yam, but citation hardening still matters and labeling must stay explicit

### Categories
1. **grains** — highest near-term calibration risk because common foods vary enormously by refinement and preparation
2. **vegetables** — highest context-quality risk because phytonutrient claims are easy to overstate without source discipline
3. **tubers** — strong need for identity cleanup before category calibration can be trusted
