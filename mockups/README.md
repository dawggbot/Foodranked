# Mockups

## visual-template-v1.html

A simple static HTML mockup of the FoodRanked visual template.

It is meant to help with:
- seeing the locked layout more concretely
- validating header/stage/subtitle/progress spacing
- confirming that the section system feels right before timing automation

It is **not** the final renderer and does not animate yet.

## Open it locally

```bash
xdg-open /home/idk/.openclaw/workspace/Foodranked/mockups/visual-template-v1.html
```

If `xdg-open` is unavailable, open it in any browser manually.
