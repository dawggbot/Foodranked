minerals are basically not part of the case here.

the upsides first: very practical staple food because cheap, common, and easy to build meals around, which matters for a tuber category built on energy foods. can be fairly filling because especially when eaten in less processed forms. brings some useful potassium because not amazing, but enough to matter a bit in the category.

now the downsides: glycemic behaviour is only middling because not as clean as the best tubers on the carb-behaviour axis. fibre is only modest because decent, but not category-leading. micronutrient return is fine rather than elite because strong enough to be respectable, not enough to be exceptional.

White Potato does have some solid upside. but the tradeoffs keep it from climbing higher. best when you want practical carbs and the rest of the profile is still reasonably clean.
