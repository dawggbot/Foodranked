# ElevenLabs Copy Pack

Use this pack for the first launch-five narration exports.

## Global settings
- speed: **1.15x**
- tone: clear, confident, fair
- energy: slightly punchy, not overhyped
- pause style: brief pause between sections, no long dramatic silences
- final line: `X TIER!` should feel like a hard stop
- use the narration block format from `production/NARRATION-OPS.md`
- speak out abbreviations like `daily value` instead of `DV`

## Export naming
Save files as:
- `production/episodes/<slug>/voice/voice-v1.mp3`
- `production/episodes/<slug>/voice/voice-v2.mp3`
- `production/episodes/<slug>/voice/voice-notes.md`

## 1) Bacon
**Script file:** `production/episodes/bacon/voice/final-narration.txt`

## Review after export
For each take, check:
- opener is immediate
- submacro words are spoken clearly
- no sentence bunches awkwardly
- final summary sounds natural aloud
- nothing comes after the tier line
