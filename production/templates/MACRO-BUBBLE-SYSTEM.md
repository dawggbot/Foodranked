# Macro Bubble System

These Aseprite-made macro bubbles are now the default visual language for the 3 macro sections.

## Current macro bubbles

- **Protein bubble** — flexed arm icon
- **Carb bubble** — lightning bolt icon
- **Fat bubble** — shield icon

## Intended meaning

- **Protein / arm** = strength, structure, recovery, body-building material
- **Carbs / lightning** = energy, speed, fuel, glycemic impact, fast-vs-stable payoff
- **Fat / shield** = protection, stability, storage, hormone / structural support, but also quality concerns

## Usage rules

- Use the macro bubble as the section's main visual badge.
- Keep one bubble visible near the macro headline during the whole section.
- Bubble should appear before or with the main macro number reveal.
- Do not let the bubble overlap subtitles.
- Keep bubble scale consistent across episodes.

## Placement guidance

Recommended placement:
- upper-left area of the section frame, near the section chip and macro headline
- large enough to read instantly at phone size
- small enough to not compete with the main stat value

## Motion guidance

- Bubble should pop in with a short scale-up and settle.
- Use a subtle hover/idle bob only if it does not create visual noise.
- On strong points or warnings, the bubble can pulse once.

## Color logic

Current bubble colors are good and should be preserved as the base identity:
- protein bubble = red family
- carb bubble = warm brown/yellow family
- fat bubble = purple/grey family

Do not recolor them per episode unless doing a deliberate alt-skin system later.

## Notes

These are not generic icons anymore.
They are part of the FoodRanked brand language.
