# Grains Audit

## Foods reviewed so far
- oats
- white-rice
- whole-wheat

## Category philosophy
Grains should reward:
- fibre density
- better glycemic behaviour
- whole-grain integrity
- useful mineral return
- practical staple value

Grains should penalize:
- refinement
- low fibre
- harsher glycemic behaviour
- weak micronutrient return

## Current strengths
- Oats already has a meaningful food-specific signal via beta-glucans.
- White rice is being punished for refinement and GI, which makes sense.
- Whole wheat is numerically set up to beat white rice clearly, which is good.

## Current problems
- White rice and whole wheat context items are too generic and repetitive.
- Several grain context items are practical/lifestyle points rather than notable nutritional distinctions.
- Oats likely deserves more exception-aware handling than the others because it may have unique compounds worth calling out.

## Food-by-food notes

### Oats
Best current positive:
- beta-glucans
Potential extra positive exception worth verifying:
- avenanthramides
Current cons are decent but need ranking:
- anti-nutrients/phytates = real and useful
- topping/preparation trap = somewhat valid, but risky if the ranked item is plain oats
- digestive tolerance issue = valid for some people, but may or may not deserve a permanent top-3 slot
Likely final direction:
- keep beta-glucans
- consider avenanthramides if evidence and project tone support it
- keep phytate caveat
- replace weakest generic con with something more structurally relevant if needed

### White rice
Current context is mostly generic practicality filler.
That should not be the main pro story.
Potential better positives:
- easy digestion / low residue / useful in certain tolerance situations
- staple practicality / culinary versatility
- lower antinutrient burden than some whole grains if relevant enough to matter
Potential better cons:
- low fibre
- weak micronutrient density
- higher GI / refinement penalty
This food should probably remain a low-tier grain unless another specific rice format changes the case.

### Whole wheat
Numerically looks much stronger than white rice.
Current context is too generic and repetitive.
Potential better positives:
- whole-grain fibre density
- stronger mineral return
- steadier carb package than refined grains
Potential better cons:
- phytates / mineral absorption caveat
- gluten tolerance issue for relevant people (must be handled carefully; not universal)
- still carb-dense and can be over-refined in common commercial formats depending on what exact whole-wheat form is being ranked

## Grains likely rewrite priority
1. white rice
2. whole wheat
3. oats polish
